Welcome to FLAX interactive language learning on mobile devices.

FLAX is an open source system that we have developed for teaching and learning a second language. It includes several language games.

For more information please visit http://flax.nzdl.org/greenstone3/flax

In Collocation Dominoes your goal is to create \‘dominoes\’ of collocations that lead on from one another. Drag and drop the partial phrases to create a match!
